<!-- GETTING STARTED -->

## Installation

1. Clone the repo
    ```sh
    git clone https://github.com/Freekson/React-Pizza.git
    ```
2. Install NPM packages
    ```sh
    npm install
    ```
3. Run command
    ```sh
    npm run start
    ```
